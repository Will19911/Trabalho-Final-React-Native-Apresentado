WeatherApp – Consulta de clima com favoritos & modo offline

Aplicativo mobile desenvolvido em React Native com Expo, que permite ao usuário consultar o clima de diferentes cidades, salvar favoritos, manter chave climática personalizada e usar em modo offline com dados mockados.

Funcionalidades

 Buscar clima por cidade
 Mostrar temperatura, umidade, sensação térmica, vento etc
 Salvar cidades como favoritas
 Persistência de dados com AsyncStorage
 Modo simulação caso não esteja conectado
 Modo offline automático
 Login simples e persistente
 Navegação com tabs personalizadas
 Digitação automática da API Key
 Ícones modernos com lucide e Feather

 Tecnologias Utilizadas

React Native (Expo)

TypeScript
